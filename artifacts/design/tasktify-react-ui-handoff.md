## Assumptions

- Gmail remains the primary onboarding path for MVP.
- Slack and Teams are optional sources after Gmail connection.
- Review, confirm, edit, and dismiss are the core actions that should dominate the UI.
- Visual styling should stay intentionally light so the structure can transfer cleanly into Figma or a production app.
