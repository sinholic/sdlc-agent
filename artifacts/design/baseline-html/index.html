Tasktify Visual Baseline

Calm, trustworthy foundations for a personal follow-up assistant.

This page translates the approved Tasktify PRD into baseline design tokens and UI behavior. It is formatted as a simple webpage so it can be imported into Figma through HTML import plugins.

Trustworthy, calm, clear, productive, supportive.

Should not feel like

Noisy, futuristic, enterprise-heavy, overly playful, or "AI magic" driven.
